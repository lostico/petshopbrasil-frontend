Como usar o Lucide Angular

Para usar o Lucide Angular em seus componentes:

1. Configure os ícones no app.config.ts (já feito):

import {{ '{' }} LucideAngularModule, Plus, Edit {{ '}' }} from 'lucide-angular';
importProvidersFrom(LucideAngularModule.pick({{ '{' }} Plus, Edit {{ '}' }}))

2. Importe o módulo no componente:

imports: [CommonModule, LucideAngularModule]

3. Use no template:

<lucide-icon name="edit" class="h-6 w-6 text-blue-500"></lucide-icon>

Ícones disponíveis:

Os seguintes ícones já estão configurados e prontos para uso:

• plus • edit • trash-2 • arrow-left • arrow-right • check • x • eye • settings

Para adicionar novos ícones, inclua-os no app.config.ts
